### Prerequisites

install.sh will install all prerequisites, but they are:

```
python3
python3-pip
python3-tk
discord.py==0.16.12
coinmarketcap==4.1.1
sortedcontainers==1.5.9
tabulate==0.8.2
pytz==2017.3
seaborn==0.8.1
```

### Installing

Very simple, run the install.sh

```
install.sh
```
